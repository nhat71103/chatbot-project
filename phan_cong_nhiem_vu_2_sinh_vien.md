
PHÂN CÔNG NHIỆM VỤ – MÔN WEB NÂNG CAO
Đề tài: Hệ thống Chatbot hỗ trợ học tập CNTT

==============================
SINH VIÊN 2 – BACKEND & DATABASE
==============================

Vai trò:
Xây dựng backend và cơ sở dữ liệu cho hệ thống chatbot.

Nhiệm vụ cụ thể:
- Xây dựng Backend bằng FastAPI.
- Thiết kế và triển khai RESTful API cho hệ thống.
- Cài đặt cơ chế xác thực JWT (Bearer Token).
- Xử lý đăng ký, đăng nhập và phân quyền người dùng.
- Thiết kế và quản lý cơ sở dữ liệu SQL Server.
- Xây dựng các bảng: users, knowledge, chat_history.
- Xử lý lưu trữ lịch sử hội thoại, pin/unpin và xóa hội thoại.
- Đảm bảo bảo mật API và kiểm soát truy cập dữ liệu.

==============================
SINH VIÊN 3 – CHATBOT & FRONTEND USER
==============================

Vai trò:
Xây dựng giao diện người dùng và logic chatbot hỗ trợ học tập.

Nhiệm vụ cụ thể:
- Thiết kế giao diện chatbot (HTML, CSS).
- Xây dựng logic frontend bằng JavaScript (SPA, fetch API).
- Gửi và nhận dữ liệu chat từ backend.
- Hiển thị lịch sử hội thoại và trạng thái chat.
- Xây dựng chatbot theo mô hình RAG đơn giản.
- Tiền xử lý câu hỏi người dùng (tokenize, loại stopword).
- So khớp câu hỏi với tri thức trong cơ sở dữ liệu.
- Xây dựng cơ chế phát hiện intent và chấm điểm câu trả lời.
- Đảm bảo trải nghiệm người dùng (UX) mượt mà.
